# LeetCode Solutions

This repository contains a collection of problems that I’ve solved on LeetCode as part of my practice.

The solutions are organized by topic to make it easier to navigate and review key patterns, such as:
- 2 Pointers
- Arrays & Strings
- Binary Search
- Hashmap & Sets
- Heaps
- Linked List
- Recursive Backtracking
- Sliding Window
- Stack
- Trees

Each folder includes implementations of problems along with my approach to solving them.  
The goal of this repo is to track progress, strengthen problem-solving skills, and build intuition for common algorithmic patterns.

More problems will be added over time.
